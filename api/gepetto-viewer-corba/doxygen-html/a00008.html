Detailed Description

template<typename _Scalar, int _Options>
class se3::SE3Tpl< _Scalar, _Options >

The rigid transform aMb can be seen in two ways:

  • given a point p expressed in frame B by its coordinate vector Bp, aMb computes its coordinates in frame A by Ap = aMb Bp.
  • aMb displaces a solid S centered at frame A into the solid centered in B. In particular, the origin of A is displaced at the origin of B: $^aM_b ^aA = ^aB$. The rigid displacement is stored as a rotation matrix and translation vector by: aMb (x) = aRb*x + aAB where aAB is the vector from origin A to origin B expressed in coordinates A.
