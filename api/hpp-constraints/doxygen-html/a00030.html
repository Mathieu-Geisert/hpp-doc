Detailed Description

Relative transformation of two fixed frames in robot joints.

The value of the function is a 6-dimensional vector. The 3 first coordinates are the position of the center of the second frame expressed in the first frame. The 3 last coordinates are the log of the orientation of frame 2 in frame 1.
