Detailed Description

Car like motion.

Implement a Reeds and Shepp motion generation on the base joint. Degrees of freedom are interpolated depending on the type of joint they parameterize: The following interpolation is made:

  • Reeds and Shepp interpolation for the base_joint_xy and base_joint_rz
  • If the joint group "reedsshepp_wheels" is present with Device::jointGroup and has 2 joints, then these joints are understood as the left and right wheel (in this order)
  • If the joint group "reedsshepp_wheels" is present with Device::jointGroup and has 1 joint, then this joints is understood as the only wheel (as a motorcycle)
  • linear interpolation for the other joints
