Package not installed

fcl does not seem to be installed

Or it has been installed more recently than this documentation. In this case, you need to configure and install again this documentation.

To install fcl, you need either
  • to get the source code through git: git clone git@github.com:humanoid-path-planner/fcl, then configure, compile and install the package.
  • install it through robotpkg: cd robotpkg/path/fcl; make update

Note that you will need to configure and install again this documentation to activate the link to fcl.
