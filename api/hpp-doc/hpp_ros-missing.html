Package not installed

hpp_ros does not seem to be installed

Or it has been installed more recently than this documentation. In this case, you need to configure and install again this documentation.

To install hpp_ros, you need either
  • to get the source code through git: git clone git@github.com:humanoid-path-planner/hpp_ros, then configure, compile and install the package.
  • install it through robotpkg: cd robotpkg/path/hpp_ros; make update

Note that you will need to configure and install again this documentation to activate the link to hpp_ros.
