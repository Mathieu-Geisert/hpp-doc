hpp-fcl Documentation

Introduction

Fcl is a library for collision detection and distance computation between various types of geometric shapes reprensented either by

  • basic shapes (fcl::ShapeBase) like box, sphere, cylinders, ...
  • or by bounding volume hierarchies of various types (fcl::BVHModel)

Using fcl

The main entry points to the library are functions

  • fcl::collide and
  • fcl::distance.
