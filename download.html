How to install HPP on your machine?

The software architecture has been tested entirely on Ubuntu 12.04 LTS, and 14.04 LTS, respectively with ros-hydro and ros-indigo. Latest developpments are developped under Ubuntu 14.04 LTS. If you are running other distributions, we expect the library to work correctly but we can not guarantee it. Follow the instructions here.
