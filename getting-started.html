The software architecture has been tested entirely on:

  • Ubuntu 12.04 LTS, with ros-hydro,
  • Ubuntu 14.04 LTS, with ros-indigo.
Current developpments are done under Ubuntu 14.04 LTS so it is preferable to choose it. If you are running other distributions, we expect the library to work correctly but we can not guarantee it. Follow the instructions here.

To run the tutorial, open a terminal and open 3 tabs by typing CTRL+SHIFT+T twice. When the terminal is selected, you can select a tab by typing ALT-[1|2|3].

Step 1 - Starting gepetto-viewer-server

In the first tab, type {% highlight bash %} $ gepetto-viewer-server {% endhighlight %} Nothing apparently happens, but the graphical user interface is ready to create a new window and to display a scene.

Step 2 - Starting hppcorbaserver

In the second tab, type {% highlight bash %} $ hppcorbaserver {% endhighlight %} See package hpp-corbaserver for details. Note that gepetto-viewer-server and hppcorbaserver executables are completely independent.

Step 3 - Controlling via a python terminal

In the third tab, type {% highlight bash %} $ python {% endhighlight %} {% highlight bash %} Python 2.7.3 (default, Feb 27 2014, 20:00:17) [GCC 4.6.3] on linux2 Type "help", "copyright", "credits" or "license" for more information. >>> {% endhighlight %} Copy-paste the content of file script/tutorial_1.py in the python terminal. A window should pop-up, displaying a scene containing a PR2 robot in a kitchen environment. See next section for details about the instructions in script/tutorial_1.py
