Motion and Manipulation planning in C++ and Python

HPP is a C++ Software Developement Kit implementing path planning for kinematic chains in environments cluttered with obstacles. Collision checking is performed by a modified version of the Flexible Collision Library developed at University of North Carolina. A visualization tool is provided to visualize obstacles, robot configurations and paths. Robots can be loaded from URDF model. Python scripting is implemented via CORBA servers and idl interfaces in a seamless way. It is a collection of software packages handled by cmake and pkg-config.
